Executive Summary

This project analyses nitrogen (N), phosphorus (P), and potassium (K) fertilizer requirements across 22 crops to identify input cost optimisation opportunities. Using Python for data processing and Google Looker Studio for interactive visualisation, the dashboard enables farmers, agronomists, and agribusinesses to compare input requirements, identify the most efficient crops by rainfall-to-input ratio, and make data-driven planting decisions that minimise unnecessary fertilizer spend.

Problem Statement

Fertilizer costs represent one of the largest operational expenses in smallholder and commercial farming. Without clear visibility into how much N, P, and K each crop requires, farmers risk over-applying inputs — driving up costs and environmental damage — or under-applying, reducing yields. This project addresses that gap by delivering a crop-by-crop input cost comparison tool with efficiency scoring built in.

Solution & Technical Implementation

  • Python Data Processing: Pandas used to clean the crop recommendation dataset, engineer a Total Input feature (N + P + K), and calculate an efficiency score (rainfall ÷ total input) per crop.
  • Looker Studio Dashboard: Interactive dashboard featuring 3 KPI scorecards (Avg N, Avg P, Avg K), a total fertilizer bar chart per crop, an efficiency ranking chart, a grouped N/P/K breakdown, and a rainfall vs input scatter chart.
  • Calculated Field: Total Input (N + P + K) created directly in Looker Studio for dynamic filtering and aggregation.
  • Interactive Filter: Crop name dropdown allowing users to isolate any of the 22 crops and update all charts simultaneously.
  • Two Data Sources: Farm Input Optimiser Clean and Efficiency Summary sheets connected and used across different charts.

Key Insights

  • Highest Input Crop: Jute requires the most total fertilizer (N+P+K), making it the most input-intensive crop in the dataset.
  • Most Efficient Crops: Coconut and orange rank highest on the efficiency score — delivering more yield per unit of input relative to rainfall.
  • Low-Input Options: Several crops like chickpea and lentil require significantly less fertilizer, offering cost-effective alternatives in suitable climates.
  • Rainfall vs Input: The scatter chart reveals no strong linear relationship — high rainfall does not always correlate with high input needs, indicating crop type dominates input decisions over climate alone.
