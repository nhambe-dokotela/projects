# Farm Profitability & Resource Tracking Tool

**Developer:** Nhambe DB

**Tool:** Microsoft Excel (Financial Modelling & VLOOKUPs)

**Sector:** Farm Management / Resource Optimization

---

## Overview

An Excel-based financial dashboard designed for large-scale farm managers and cooperatives to track the relationship between input costs and crop yields. By centralising financial data into a single unified view, the tool identifies "capital leaks" and ensures farms remain financially sustainable across seasons.

---

## Problem Statement

Many farmers struggle to see the true profitability of individual crops because expenses (seeds, fertilizer, labour) are tracked separately from revenue. Without a unified view it is difficult to determine which crops are generating profit and which are operating at a loss, leading to poor planting decisions and wasted resources.

---

## Solution & Technical Implementation

- **Integrated Data Streams:** Relational model linking the Input Log (Expenses) with the Yield Log (Revenue) via crop and date identifiers
- **Variance Analysis:** Budgeted vs. Actual tracking system to identify overspending in real-time across all cost categories
- **Profit Margin Visualisation:** Charts ranking crops (e.g., Tomatoes vs. Cabbage) by actual Gross Profit Margin - not just tonnage
- **Farm Selector:** Dropdown-driven dashboard that filters all visuals and KPIs by selected farm - each farm tracks 3 crops, zeros reflect crops not planted on that farm

---

## Key Impact

- **Financial Sustainability:** Identifies capital leaks and gives managers a clear path to improving the bottom line
- **Strategic Planting:** Historical ROI data guides future crop arrangements and seasonal planning
- **Resource Management:** Better allocation of labour and inputs across provinces and farm sites

## How to Use the Dashboard

1. Download `Farm_Profitability_Dashboard.xlsx`
2. Open with **Microsoft Excel**
3. Use the **Farm Selector** dropdown to switch between farms
4. All KPI cards, charts and profit margin visuals update automatically
