# Crop Yield Predictor

Predicts global crop yield using rainfall, temperature and pesticide data.

# Tools:
Python (Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn) & Excel

## Dataset
Kaggle - Crop Yield Prediction Dataset avalable @ https://www.kaggle.com/datasets/patelris/crop-yield-prediction-dataset

## Model Results
| Model | RMSE | R²|
|------|-------|----|
| Linear Regression |  81,501.76 | 0.08 |
| Random Forest | 10,181.76 | 0.99 |

## Key Findings
- Highest yielding crop: Potatoes
- Highest yielding country: Belgium
- Random Forest significantly outperforms Linear Regression
